There is a simple program for monitoring offers for renting apartments and rooms on the [Kufar.by]
<hr/>

## Create your own Telegram bot
> How to [create] Telegram bot
<br/>

## Create an .env file
> Create an `.env`:

```sh
MONGO_DB_URL=...
TELEGRAM_BOT_TOKEN=...
CHAT_ID=...
KUFAR_URL=...
INTERVAL_DELAY_MS=...
MAX_RENTAL_SAVED=...

```
<br/>

## Start project
```sh
make build-no-cache
make up
```
